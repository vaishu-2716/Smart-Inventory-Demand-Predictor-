# Smart Inventory Demand Predictor 📊

## 📌 Overview
This project predicts future product demand using historical sales data. It helps businesses reduce overstocking and avoid shortages using data-driven insights.

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## 🚀 Features
- Forecasts future sales using Linear Regression
- Visualizes actual vs predicted demand
- Helps improve inventory planning

## 📂 Project Files
- main.py → Python code for prediction model
- sales.csv → Dataset used for training
- screenshot.png → Output graph
