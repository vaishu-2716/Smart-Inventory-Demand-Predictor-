# 📦 Smart Inventory Demand Predictor (Streamlit)

An interactive web application that predicts product demand using historical sales data. Built using **Python and Streamlit**, this project demonstrates how data-driven forecasting can help businesses reduce overstocking and avoid shortages.

---

## 🚀 Live Demo

👉 https://smartinventorydemandpredictor.streamlit.app/

---

## 📌 Project Overview

Efficient inventory management is essential for businesses to minimize costs and meet customer demand. This project allows users to:

* Upload historical sales data
* Analyze trends over time
* Predict future demand
* Make informed inventory decisions

It simulates real-world demand forecasting systems used in retail and supply chain management.

---

## ⚙️ Tech Stack

* **Python** – Core programming
* **Streamlit** – Web application framework
* **Pandas** – Data processing and analysis
* **CSV Dataset** – Input sales data

---

## ✨ Features

* 📂 Upload custom CSV dataset
* 📊 View and explore sales data
* 📈 Demand prediction visualization
* ⚡ Interactive and easy-to-use interface
* 🌐 Fully deployed web application

---

## 📊 Prediction Logic

* Uses **rolling average method** to estimate demand
* Demonstrates basic time-series forecasting concepts

## 🛠️ How to Run Locally

1. Clone the repository:

```id="2h1k3l"
git clone https://github.com/vaishu-2716/Smart-Inventory-Demand-Predictor.git
cd Smart-Inventory-Demand-Predictor
```

2. Install dependencies:

```id="q1p9zs"
pip install -r requirements.txt
```

3. Run the application:

```id="7mjx3v"
streamlit run app.py
```
